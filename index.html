Mr. Gibberson's Classes

Together we will learn programming!

We are thrilled to embark on a new educational adventure, as we firmly believe that children are the architects of our future. At our school, we are passionate about empowering our students with knowledge and skills that will open up endless opportunities for them. One of the most exciting and valuable areas we are eager to explore with our students is the world of programming. Programming is not only immensely fun but also incredibly useful in today's technology-driven world. That's why we are preparing a series of engaging and dynamic classes where kids can dive into the fascinating realm of coding.

PHP
We will learn what PHP is, how it works, and why it was invented. We are going to find out how to use PHP to connect and communicate with databases!
Github
We will learn what GitHub is, why it is so useful, and how to host our websites on the internet with GitHub, just like this website.
Javascript
We will learn how useful JavaScript is, why it is used, and why it is disliked by many programmers.
Python
In this class, we will work with an API using Python. We will learn how useful API keys are through one of the coolest projects: creating and posting TikTok videos, all with Python.
HTML & CSS
We will learn how to make websites with HTML and then how to make them look beautiful with CSS. We will discover what it takes to create a great website.
Tiktok
The most exiting project will be TikTok. we will learn how to work with the TikTok API. We will create the full image. A UI where you can create video's with just one click on the button. And then it will be automaticly uploaded to tiktok!
